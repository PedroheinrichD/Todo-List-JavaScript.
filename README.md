# 📝 To-Do List com JavaScript

Aplicação simples de lista de tarefas desenvolvida com **HTML, CSS e JavaScript puro**, focada em manipulação de DOM e eventos de teclado.

## 🚀 Funcionalidades

- Adicionar tarefas pressionando **Enter**
- Marcar tarefas como concluídas com **checkbox**
- Alteração visual ao concluir tarefas
- Criação dinâmica de elementos no DOM

## 🧠 Conceitos praticados

- Manipulação de DOM
- Eventos de teclado (`keyup`)
- `createElement`
- Estruturação básica de interface
