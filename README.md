# To-Do-List-com-JavaScript-Manipula-o-de-DOM-
Aplicação simples de lista de tarefas desenvolvida com HTML, CSS e JavaScript puro, focada em manipulação de DOM e eventos de teclado.  O usuário pode adicionar tarefas pressionando Enter, marcar tarefas como concluídas utilizando checkbox, e visualizar a alteração visual quando a tarefa é completada.
